# API Setup Instructions

## ExerciseDB API Setup

1. **Получите RapidAPI ключ:**
   - Зайдите на [RapidAPI ExerciseDB](https://rapidapi.com/justin-WFnsXH_t6/api/exercisedb)
   - Зарегистрируйтесь или войдите в аккаунт
   - Подпишитесь на API (есть бесплатный тариф)
   - Скопируйте ваш RapidAPI ключ

2. **Настройте ключ в приложении:**
   - Откройте файл `.env` в корне проекта
   - Замените значение `RAPIDAPI_KEY` на ваш ключ:
     ```
     RAPIDAPI_KEY="ваш_ключ_здесь"
     ```

3. **Проверьте работу:**
   - Запустите приложение: `flutter run`
   - В тренировках должны загружаться GIF анимации упражнений
   - Если API не работает, приложение автоматически переключится на локальные данные

## Возможные проблемы и решения

### Ошибка 403 "You are not subscribed to this API"
- Убедитесь, что вы подписались на API в RapidAPI
- Проверьте, что ключ скопирован правильно
- Убедитесь, что не превышен лимит запросов

### Отсутствуют медиа в упражнениях  
- Приложение автоматически использует локальные данные
- Добавьте ваши GIF файлы в `assets/gifs/exercises/`
- Обновите fallback данные в `workout_media_service.dart`

### Приложение крашится при загрузке упражнений
- Теперь исправлено: все API ошибки обрабатываются gracefully
- Приложение переключается на локальные данные при ошибках

## Тестирование

Для тестирования без API ключа:
1. Оставьте `RAPIDAPI_KEY="demo-key"` в .env
2. Приложение будет использовать только локальные данные
3. Все функции кроме загрузки внешних GIF будут работать
