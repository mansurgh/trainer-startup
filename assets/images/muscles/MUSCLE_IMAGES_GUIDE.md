# Изображения мышц для SexyWorkoutScreen

## Требования к изображениям

### Размеры и формат
- **Размер**: 400x400 пикселей (1:1 соотношение)
- **Формат**: PNG с прозрачным фоном
- **DPI**: 72-150 для оптимальной производительности
- **Размер файла**: до 100KB на изображение

### Стиль и дизайн
- **Фон**: Прозрачный (PNG с альфа-каналом)
- **Стиль**: Минималистичные силуэты человека
- **Цвета**: 
  - Основная часть тела: серый (#808080) или белый с прозрачностью 30-40%
  - Выделенные мышцы: яркий акцентный цвет (#007AFF, #00D4AA и т.д.)
- **Толщина линий**: 2-3px для контуров
- **Детализация**: Средняя (не слишком детально, но понятно)

### Список необходимых изображений

#### Передняя проекция (Front)
1. **chest.png** - выделенные грудные мышцы
2. **front_delts.png** - передние дельтовидные мышцы
3. **side_delts.png** - боковые дельтовидные мышцы
4. **biceps.png** - бицепсы
5. **abs.png** - мышцы пресса
6. **quadriceps.png** - четырехглавые мышцы бедра
7. **calves_front.png** - икроножные мышцы (вид спереди)

#### Задняя проекция (Back)
1. **back.png** - широчайшие мышцы спины
2. **rear_delts.png** - задние дельтовидные мышцы
3. **triceps.png** - трицепсы
4. **glutes.png** - ягодичные мышцы
5. **hamstrings.png** - задняя поверхность бедра
6. **calves_back.png** - икроножные мышцы (вид сзади)
7. **traps.png** - трапециевидные мышцы

### Примеры использования в коде

```dart
// В SexyWorkoutScreen
Widget _buildMuscleImage(String muscleGroup, bool isHighlighted) {
  return Container(
    width: 200,
    height: 200,
    child: Stack(
      children: [
        // Базовое изображение тела
        Image.asset(
          'assets/images/muscles/body_base.png',
          fit: BoxFit.contain,
          color: Colors.white.withOpacity(0.3),
        ),
        // Выделенная мышца
        if (isHighlighted)
          Image.asset(
            'assets/images/muscles/$muscleGroup.png',
            fit: BoxFit.contain,
            color: const Color(0xFF007AFF),
          ),
      ],
    ),
  );
}
```

### Альтернативные решения

Если создание кастомных изображений занимает много времени, можно использовать:

1. **Иконки Font Awesome** или **Material Icons**
2. **SVG иконки** из библиотек типа Heroicons, Feather Icons
3. **Готовые muscle group icons** из фитнес приложений

### Инструменты для создания
- **Figma** - для векторного дизайна
- **Adobe Illustrator** - профессиональный векторный редактор  
- **Canva** - простой онлайн редактор
- **Procreate** (iPad) - для рисования силуэтов

### Оптимизация
- Используйте **flutter_svg** для SVG файлов
- Кэшируйте изображения с помощью **cached_network_image**
- Предзагружайте изображения в initState() для плавной анимации
